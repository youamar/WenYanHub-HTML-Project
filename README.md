# 📜 WenYanHub HTML Project

> **Developed collaboratively by:** Chang Kai Yang, Gan Zhi He, Nicholas Wong Heng, and Tan Bi Jin.

WenYanHub is an interactive web-based portal designed to facilitate the learning of Classical Chinese. This repository contains the front-end layout and server-side logic developed using ASP.NET Web Forms and web technologies.

## 🌟 Features
* **Interactive Learning Interface**: User-friendly web pages designed for exploring and learning Classical Chinese.
* **Bilingual Support**: Developed with the goal of providing a seamless learning experience for users reading in both modern and classical contexts.
* **Dynamic Web Pages**: Integration of front-end web design with back-end logic for a responsive and engaging user experience.

## 💻 Technologies Used
* **Framework**: ASP.NET Web Forms
* **Back-end Language**: C#
* **Front-end**: HTML, CSS, JavaScript
* **Development Environment**: Visual Studio

## 📂 Project Structure
* `WenYanHub/` - The main directory containing the web pages (.aspx), styling files (.css), scripts (.js), and back-end logic (.cs).
* `WenYanHub.sln` - The Visual Studio solution file to open and run the entire project.
